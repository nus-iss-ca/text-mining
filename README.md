# GIT
Instruction for git commands:
- Pull source code from repo
```sh
$ git pull origin master
```
- Check file status
```sh
$ git status
```
- Add file after modify
```sh
$ git add .
```
- Commit file to repo`
```sh
$ git commit -m "message"
```
- Push to repo
```sh
$ git push origin master
```
